# Threads to VM
---
- ## VM escape
	- an attacker manage to break out of a normally isolated VM by interacting directly with the [[hypervisor]]
- ## Data Remnants
	- Contents of a virtual machine that exist as deleted files
- ## Privilege Elevation
	- user is able to grant itself the ability to run functions as a higher level user 
- ## Live migration between VMs. 
	- the attacker place it's self in the middle of data transfer from one VM to another.
